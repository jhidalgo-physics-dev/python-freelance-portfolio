# Python Freelance Portfolio

This repository contains Python projects demonstrating data analysis, web scraping, and automation workflows.

## Projects

### Stock Analysis
Downloads historical stock data and generates technical analysis charts.

Tools used:
- Pandas
- Matplotlib
- yfinance

### Web Scraper
Scrapes structured data from a website and exports it to CSV.

Tools used:
- Requests
- BeautifulSoup
- Pandas

### Data Cleaner
Automates cleaning messy datasets including duplicate removal and missing value handling.

Tools used:
- Pandas
